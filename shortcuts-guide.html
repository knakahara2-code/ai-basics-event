AIを使うときに便利な
ショートカット集

スクリーンショット・コピペ・音声入力 — これだけ覚えればAI活用が10倍はかどる

🎯 なぜショートカットが大事？
AIに画像を見せたい → スクショ → 貼り付け。
AIの回答を使いたい → コピー → 貼り付け。
長い文章を打つのが面倒 → 音声入力。
この3つの流れを覚えるだけで、AIとのやりとりが劇的にスムーズになります。

Windows
Windows 10 / 11 共通
📸
スクリーンショット（範囲選択）
Win + Shift + S
画面の好きな部分だけを切り取ってコピー。撮ったらすぐAIのチャットに貼り付けられる。
💡 エラー画面やWebページの一部をAIに見せたいときに最強！
📋
コピー
Ctrl + C
選択した文字・画像をクリップボードにコピー。
📌
貼り付け（ペースト）
Ctrl + V
コピーした文字・画像をAIのチャット欄などに貼り付ける。
💡 スクショを撮った後、AIのチャット欄でCtrl+Vすれば画像がそのまま送れる！
🔄
ウィンドウ切り替え
Alt + Tab
開いているアプリを瞬時に切り替え。AIとExcelを行ったり来たりするときに便利。
🎤
音声入力
Win + H
キーボードを打たずに、声で文字入力。AIへの長い指示も話すだけ。
💡 「句読点」「改行」も声で入力できる。慣れると超快適！
✂️
全選択
Ctrl + A
ページ全体のテキストを一括選択。長い文章をまるごとAIに貼り付けたいときに。
↩️
元に戻す
Ctrl + Z
間違えて消してしまったときに。直前の操作を取り消す。
🚀 よくある活用パターン
エラー画面をAIに聞く： Win+Shift+S → エラー部分をスクショ → AIのチャットで Ctrl+V → 「これ何？」
AIの回答をExcelに貼る： AIの回答を選択 → Ctrl+C → Alt+TabでExcelに切替 → Ctrl+V
長い指示を声で入力： AIのチャット欄をクリック → Win+H → 話す → 送信

💡 初回は設定 → キーボード → 音声入力 をオンにする必要あり

iPhone / iPad
iOS / iPadOS
📸
スクリーンショット
Face ID搭載機： サイドボタン + 音量↑ 同時押し
ホームボタン搭載機： サイドボタン + ホームボタン 同時押し
左下にプレビューが表示される。タップすると編集・切り取りも可能。
📋
コピー＆貼り付け
コピー：テキストを長押し → 範囲選択 → 「コピー」をタップ
貼り付け：入力欄を長押し → 「ペースト」をタップ
💡 写真アプリでスクショを開いて「共有」→ AIアプリに直接送ることもできる
🎤
音声入力
キーボードの 🎤 マイクボタン（スペースキーの横）をタップ → 話す → もう一度タップで終了
💡 ChatGPTアプリ・Claudeアプリなら、アプリ内のマイクボタンで直接音声会話もできる！
🔍
画像内のテキストをコピー（Live Text）
写真の中の文字を長押し → テキストとして選択・コピーできる。書類や看板の文字をAIに送るときに便利。
💡 iPhone 15以降 / iOS 15以降で利用可能

ほぼ全機種共通。撮影後に通知バーから編集・共有ができる。

貼り付け：入力欄を長押し → 「貼り付け」をタップ

カメラアプリ → Googleレンズモード → テキストを映す → 選択してコピー。書類の文字をAIに送るのに便利。
